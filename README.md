captureandsaver-cli
===================
## What's this?
起動した瞬間のスクリーンショットを取得し、デスクトップに保存します。  
ファイル名は```Screenshot-YYYY-MM-DD_HH-MM-SS.png```で固定されます。  
CLIという名前が付いていますが、**現時点でオプションなどは実装されていません。**
